# Lab Event Talks Application

This repository contains a single-page website for a 1-day event filled with technical talks. The site displays the schedule, including talk timings, speakers, categories, and descriptions. Users can also search for talks based on their category.

## Features

-   **Dynamic Schedule:** Displays a full day's schedule of technical talks, including 1-hour talk slots, 10-minute transitions, and a 1-hour lunch break.
-   **Talk Details:** Each talk includes a title, speaker(s), category keywords, and a description.
-   **Category Search:** Users can filter the displayed talks by typing keywords into a search bar.
-   **Single-File Deployment:** The entire website is compiled into a single `index.html` file, making it easy to deploy to any static file server.
-   **Themed Content:** Placeholder talks are themed around "The Meaning of Life."

## Technologies Used

-   **Frontend:** HTML, CSS, JavaScript (client-side)
-   **Build Tool:** Node.js (for bundling HTML, CSS, and JS into a single file)

## Local Setup and Run Instructions

To get a local copy up and running, follow these simple steps.

### Prerequisites

-   Node.js (for running the build script)
-   Python 3 (for running a simple local web server)

### Build the Website

The website's source code (`index.html`, `style.css`, `script.js`) is bundled into a single `dist/index.html` file using a Node.js script.

1.  **Ensure Node.js is installed.**
2.  **Run the build script:**
    ```bash
    node build.js
    ```
    This will create a `dist` directory with the `index.html` file inside.

### Serve the Website Locally

After building the website, you can serve it using Python's built-in HTTP server:

1.  **Navigate to the project root directory** (where `dist` folder is located).
2.  **Start the server:**
    ```bash
    python3 -m http.server 8000 --directory dist
    ```
    If `python3` doesn't work, try `python -m http.server 8000 --directory dist`.

3.  **Open your web browser** and go to `http://localhost:8000`.

You should now see the event schedule. Use the search bar to filter talks by category (e.g., "Philosophy", "Neuroscience").

## File Structure

-   `index.html`: The main HTML structure (before bundling).
-   `style.css`: The CSS styles for the website.
-   `script.js`: The JavaScript logic for generating the schedule and handling search, along with the talk data.
-   `build.js`: A Node.js script to bundle `index.html`, `style.css`, and `script.js` into a single output file.
-   `dist/index.html`: The final, self-contained website file generated by `build.js`.
-   `.gitignore`: Specifies files and directories that Git should ignore.
